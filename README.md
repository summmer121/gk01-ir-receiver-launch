# GK01 红外收发器（ESPHome 固件）

基于 **ESP12F** 芯片的红外学习 + 收发器，支持 **NEC**、**RC-SWITCH**、**RAW** 协议。通过 **ESPHome API** 接入 Home Assistant，同时支持 **MQTT Discovery** 动态创建按钮实体。

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 红外接收 | GPIO5，支持 NEC / RC-SWITCH / RAW，自动识别 |
| 红外发射 | GPIO14，支持 NEC 和 RAW 发射 |
| 自定义映射 | 接收红外码 → 映射自定义值 → 写入 keymap → 触发 HA 自动化 |
| 动态学习 | 接收信号 → 填写名称 → 学习发送写入 → HA 自动创建按钮实体 |
| MQTT Discovery | 学习的每个按钮通过 `homeassistant/button/` 自动注册到 HA |
| MQTT 配置持久化 | 地址/端口/用户名/密码保存到 Flash，重启不丢失 |
| 状态指示 | 黄灯 = 接收信号，红灯 = 发送信号 |

---

## 硬件接线

| 功能 | GPIO | 说明 |
|------|------|------|
| 红外接收 | GPIO5 | 带内部上拉，inverted |
| 红外发射 | GPIO14 | 载波 38kHz |
| 红灯 | GPIO12 | 发送时闪烁 |
| 黄灯 | GPIO13 | 接收时闪烁 |
| 按钮 | GPIO16 | D0，下拉 |

## 界面说明（web 端 192.168.x.x）

从上到下：

1. 运行时间 / WiFi 信号
2. 启动信息 / **MQTT 连接状态**
3. 槽位数量 / 发送槽位列表 / 接收映射列表
4. 红外事件计数器 / 红外码状态
5. MQTT 地址 / 用户名 / **密码（隐藏）** / 端口
6. 红外码值 / 映射值(发送值)
7. 接收映射写入 / 学习发送写入 / 删除发送按键 / 删除映射 / 重启

## 使用流程

### 接收映射（触发 HA 自动化）

1. 按遥控器 → 红外码自动填入「红外码值」
2. 在「映射值」输入自定义名称（如 `living_room_light`）
3. 点击「接收映射写入」
4. 后续收到该红外码，HA 会收到 `gk01/ir/mapped` MQTT 消息

### 学习发送按键（HA 中生成按钮）

1. 按遥控器 → 红外码自动填入「红外码值」
2. 在「映射值」输入按钮名称（如 `空调开关`）
3. 点击「学习发送写入」
4. HA 自动出现同名按钮，点击即可发射该红外信号

### 删除

- 「删除发送按键」：填红外码或名称 → 删除并清除 HA 实体
- 「删除映射」：填红外码或映射值 → 删除 keymap

---

## 接入 Home Assistant

### 方式一：ESPHome 集成（API）

HA 设置 → 集成 → 添加 ESPHome → 输入设备 IP → 自动发现所有实体

### 方式二：MQTT 集成（动态按钮）

确保 HA 已配置 MQTT 集成。学习的按钮自动通过 `homeassistant/button/gk01_ir_*/config` 注册。
